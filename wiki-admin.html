Demarrage rapide

  1. Ouvrir l'application puis se connecter avec le code d'acces bureau.
  2. Verifier en haut de l'admin la saison publique du formulaire : c'est la saison visible par les familles.
  3. Consulter l'onglet Liste pour voir les nouveaux dossiers.
  4. Pendant les permanences, utiliser la partie Permanence pour encaisser, verifier les pieces et valider.
  5. Utiliser Exports seulement quand un fichier Excel est necessaire.
Le numero de licence FFF sert a retrouver une personne deja connue dans la base Footclubs et peut pre-remplir les champs disponibles.

Statuts des dossiers

En attente
Dossier recu, pas encore traite en permanence.
Incomplet
Il manque une piece, une information ou un paiement.
Valide
Dossier regle et valide. Dans l'application, valide veut dire paye.
Refuse
Dossier a ne pas poursuivre ou inscription annulee.

Le changement de statut s'enregistre automatiquement. Si une personne du bureau change un statut, la page permanence doit se mettre a jour automatiquement aussi, a condition que les deux pages utilisent la meme saison.

Permanence licence

Objectif

La permanence sert a retrouver vite un dossier, verifier les informations du membre, suivre les pieces fournies, ajouter si besoin des articles boutique achetes pendant la permanence, puis valider quand tout est regle.

Bon deroulement

  1. Rechercher la personne par nom, prenom, reference ou famille.
  2. Verifier la categorie, le tarif, le mode de paiement et les pieces.
  3. Pour un paiement en plusieurs fois, verifier les dates d'encaissement.
  4. Si tout est bon, passer le statut a Valide.
Quand le statut licence passe a Valide, les achats boutique faits pendant la permanence passent automatiquement en regle.

Boutique club

La boutique est reservee a la partie bureau / secretariat. Elle ne s'affiche pas dans la preinscription famille.

Produits

  • Ajouter ou modifier les articles depuis l'onglet Boutique > Produits.
  • Chaque produit peut avoir une categorie, un prix, des tailles et une photo.
  • Les categories boutique permettent de regrouper textile, equipement joueur, accessoires, commandes speciales, etc.

Commandes

  • Les achats faits pendant une permanence peuvent etre inclus dans le paiement licence.
  • Les commandes ajoutees plus tard dans la saison restent separees du paiement licence.
  • Suivre les statuts : a regler, regle, commande, en attente fournisseur, recu, livre, annule.
  • La visualisation par categorie joueur aide a preparer les commandes groupees.

Exports

L'onglet Exports regroupe les fichiers Excel utiles : toutes les preinscriptions, equipes, paiements, tailles, certificats, contacts, base licencies et boutique.

Conseil : filtrer d'abord dans l'admin si vous voulez regarder un groupe precis, puis exporter le fichier correspondant.

Configuration admin

Saison

La saison choisie en haut de l'admin est la saison de travail. La saison publique est celle visible par les familles dans le formulaire.

Tarifs et remises

Les tarifs, remises famille, permanences, pieces a fournir, codes d'acces et parametres boutique sont modifiables dans l'admin.

Certificats

L'onglet Certifs distingue les joueurs detectes depuis la base Footclubs et les joueurs deja preinscrits qui necessitent un certificat medical.

Depannage

  • Si un dossier n'apparait pas, verifier la saison active dans les deux pages.
  • Si la synchronisation indique hors-ligne, les donnees restent sauvegardees localement et se resynchronisent quand Firebase redevient disponible.
  • Utiliser le bouton Diagnostic de l'admin pour comparer donnees locales, donnees Firebase et dossiers affiches.
  • Si deux personnes travaillent en meme temps, garder l'application ouverte sur la meme saison pour recevoir les mises a jour.
