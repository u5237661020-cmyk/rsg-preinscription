==========================================
  RSG - MODE D'EMPLOI INSTALLATION
==========================================

Lancez les scripts DANS L'ORDRE en double-cliquant :

  1-VERIFIER.bat
  2-INSTALLER-DEPENDANCES.bat
  3-PUBLIER-GITHUB.bat
  4-DEPLOYER.bat

Si l'un plante, vous savez exactement lequel.
Si tout va bien, ne lancez plus les scripts 1 et 2.
Pour les mises a jour, utilisez UPDATE-WINDOWS.bat

==========================================
  AVANT DE COMMENCER
==========================================

1) Compte GitHub : https://github.com/signup

2) Token GitHub :
   - Aller sur https://github.com/settings/tokens/new
   - Note : "RSG App"
   - Expiration : 1 an
   - Cocher la case "repo"
   - Generate token
   - COPIER le token (il ne sera plus jamais affiche)

3) Creer le depot GitHub :
   - Aller sur https://github.com/new
   - Repository name : rsg-preinscription
   - Public (coche)
   - NE PAS cocher "Add a README file"
   - Create repository

==========================================
  EN CAS DE PROBLEME
==========================================

"Le script bloque sans message" :
  - Fermez la fenetre noire
  - Relancez le script qui bloquait
  - Notez le DERNIER message affiche avant le blocage

"Authentication failed" au push :
  - Vous avez utilise votre mot de passe au lieu du Token
  - Refaites un Token (etape 2) et reessayez

"Repository not found" :
  - Le depot n'existe pas sur GitHub
  - Soit le nom est mal orthographie
  - Soit vous ne l'avez pas encore cree
